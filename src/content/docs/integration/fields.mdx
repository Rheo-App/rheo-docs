## PUT /integration/v1/items/:external_id — request fields

| Field | Type | Required | Description |
|---|---|---|---|
| `title` | string | Yes | Item title. Max 255 characters. |
| `description` | string | No | Plain text description. |
| `price` | number | Yes | Starting price in SEK. Minimum 1. |
| `shippingCost` | number | No | Flat shipping cost in SEK. Use `0` for free shipping. |
| `weightKg` | number | No | Item weight. Used for shipping calculations. |
| `currency` | string | No | Defaults to `SEK`. |
| `imageUrls` | string[] | Yes | HTTPS URLs to item images. Max 10. Also accepted as `images` or `image_urls`. |
| `externalReference` | string | No | Your internal ID for this item. Stored and echoed back in responses. |
| `domain` | object | No | Domain-specific attributes. See [Domain System](/integration/domains/). |
| `schemaVersion` | string | No | Semver version of the domain schema used, e.g. `"1.0.0"`. Defaults to `"1.0.0"` if omitted. Stored with the item for future schema registry enforcement. |
| `autoPublishTradera` | boolean | No | Publish a Tradera auction when images are ready. Only applied on item creation. Default: `false`. |
| `useAiEnhancement` | boolean | No | Run the item through Rheo AI to improve title and description. Default: `false`. |

## GET /integration/v1/items/:external_id — response fields

| Field | Description |
|---|---|
| `externalId` | The ID you supplied on creation. |
| `rheoItemId` | Rheo's internal UUID for the item. |
| `rheoStatus` | `draft`, `active`, `sold`, `not_listed`, or `archived`. |
| `traderaStatus` | `active`, `sold`, `ended`, `pending_sync`, or `sync_error`. |
| `traderaAdId` | Tradera's internal ad ID. `null` if not yet published. |
| `traderaAdUrl` | Direct link to the live ad. `null` if not published. |
| `price` | Current price in SEK. |

---

## Item status values

| Value | Meaning |
|---|---|
| `draft` | Received by Rheo, images not yet downloaded. |
| `active` | Images processed, item live in Rheo marketplace. |
| `sold` | Sold on Tradera or Rheo. |
| `not_listed` | De-listed manually or via PATCH. |
| `archived` | Deleted while a Rheo order was in progress. |
| `pending_verification` | Awaiting identity or payment verification. |
| `pending_review` | Flagged for manual review. |
| `rejected` | Rejected after review. |
| `identified` | Identified as a specific item by AI but not yet listed. |

## Shipping

| Field | Type | Description |
|---|---|---|
| `shippingClass` | string | `letter`, `small`, `medium`, `large`, `xlarge`, `bulky`, `freight`. Used to select a carrier rate. |
| `weightKg` | number | Weight in kg. |
| `lengthCm` | number | Length in cm. |
| `widthCm` | number | Width in cm. |
| `heightCm` | number | Height in cm. |
| `dealType` | string | `ship`, `pickup`, or `both`. |

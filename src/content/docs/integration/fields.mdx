## Domain object

The `domain` field carries item-type-specific attributes. It uses a tagged union — the `domain` property inside the object determines the type.

---

## Automotive parts

Use `"domain": "auto_parts"` for spare parts, body panels, engines, gearboxes, and parts from motorcycles, ATVs, snowmobiles, and caravans.

```json
{
  "domain": {
    "domain": "auto_parts",
    "oemCode": "31400452",
    "partName": "Bromsok fram vänster",
    "manufacturer": "Volvo",
    "compatibleModels": ["XC90", "XC90 II"],
    "donorVehicleYear": 2018,
    "vehicleType": "Bil",
    "conditionGrade": "B"
  }
}
```

| Field | Type | Description |
|---|---|---|
| `oemCode` | string | OEM part number. Optional — `partName` alone is sufficient. Shown in the Tradera ad. |
| `partName` | string | Human-readable part name (e.g. `"Bromsok fram vänster"`, `"Baklucka"`). |
| `manufacturer` | string | Vehicle manufacturer (e.g. `Volvo`, `BMW`, `Kawasaki`). |
| `compatibleModels` | string[] | List of compatible model names. |
| `donorVehicleYear` | integer | Model year of the donor vehicle. |
| `vehicleType` | string | Vehicle type the part came from. See values below. |
| `conditionGrade` | string | Condition rating. Rheo accepts any string; recommended values below. |

### Vehicle types

| Value | Meaning |
|---|---|
| `Bil` | Passenger car (default if omitted) |
| `MC` | Motorcycle |
| `Skoter` | Snowmobile |
| `ATV` | ATV / quad bike |
| `Husvagn` | Caravan / trailer |
| `Moped` | Moped |
| `Traktor` | Tractor |

### Condition grades

| Grade | Meaning |
|---|---|
| `A*` | New old stock or unused |
| `A` | Excellent — minimal wear, fully functional |
| `B` | Good — normal wear, fully functional |
| `C` | Fair — visible wear or minor damage, functional |

These grades are displayed verbatim in the Tradera ad description.

## Electronics

```json
{
  "domain": {
    "domain": "electronics",
    "serialNumber": "SN123456",
    "memoryGb": 256,
    "batteryHealthPercentage": 88
  }
}
```

| Field | Type | Description |
|---|---|---|
| `serialNumber` | string | Device serial number. |
| `memoryGb` | integer | Storage capacity in GB. |
| `batteryHealthPercentage` | integer | Battery health (0–100). |

## Fashion

```json
{
  "domain": {
    "domain": "fashion",
    "size": "M",
    "material": "Cotton",
    "gender": "unisex"
  }
}
```

| Field | Type | Description |
|---|---|---|
| `size` | string | Size label (e.g. `M`, `42`, `UK 8`). |
| `material` | string | Primary material. |
| `gender` | string | `mens`, `womens`, or `unisex`. |

## Vehicles

Use `"domain": "vehicles"` for whole vehicles being sold (cars, motorcycles, boats, caravans).

```json
{
  "domain": {
    "domain": "vehicles",
    "registrationPlate": "ABC123",
    "make": "Volvo",
    "model": "XC90",
    "year": 2018,
    "mileageKm": 94000,
    "fuelType": "diesel",
    "color": "black"
  }
}
```

| Field | Type | Description |
|---|---|---|
| `registrationPlate` | string | Swedish registration plate. Rheo can auto-fill make/model/year from it via the plate lookup endpoint. |
| `make` | string | Vehicle brand. |
| `model` | string | Model name. |
| `year` | integer | Model year. |
| `mileageKm` | integer | Odometer reading in kilometres. |
| `fuelType` | string | `petrol`, `diesel`, `electric`, or `hybrid`. |
| `color` | string | Body colour. |

## Plate lookup

Resolve a Swedish registration plate to vehicle metadata:

```
GET https://market.rheo.se/vehicles/lookup?plate=ABC123
```

```json
{
  "registrationPlate": "ABC123",
  "make": "Volvo",
  "model": "XC90",
  "year": 2018,
  "fuelType": "diesel",
  "color": "black"
}
```

Results are cached for 24 hours. Returns `404` if the plate is not found.

---

## Generic items

Omit the `domain` field entirely for items that do not fit the above categories.

---

## How domain data appears in Tradera ads

Rheo appends a structured section at the end of the Tradera ad description:

```
--- Technical Specification ---
OEM: 31400452
Condition: B
Compatible with: Volvo XC90 (2018)
```

This section is generated automatically from the `domain` fields you provide. The more data you send, the better the ad performs in Tradera search.
